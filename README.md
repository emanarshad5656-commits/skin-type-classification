# Skin Type Classification Using Deep Learning

This project classifies:
- Dry Skin
- Normal Skin
- Oily Skin

## Technologies Used
- Python
- TensorFlow
- MobileNetV2
- Gradio
- Google Colab

## Features
- Deep Learning based prediction
- Transfer Learning
- Web Interface
- High Accuracy
